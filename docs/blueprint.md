# **App Name**: AgroVision AI

## Core Features:

- AI Plant Disease Detection: Detect plant diseases from uploaded images using AI. Outputs plant name, disease, confidence level, symptoms, causes, and treatment (organic + chemical).
- Multilingual Indian Language Support: Support for multiple Indian languages (Hindi, Telugu, Tamil, Kannada, Malayalam, Marathi, Bengali) with voice input/output.
- Voice-Enabled AI Chatbot: AI chatbot providing crop care advice, fertilizer guidance, and seasonal farming tips, accessible via voice and text in multiple Indian languages.
- Google Authentication: Secure user authentication using Google Sign-In and email/password options via Firebase Authentication.
- Detection History Storage: Store and manage all plant disease detection records with date-wise history and crop-wise filtering using Firestore.
- Daily Farming Tip: Provide AI generated farming tips to the user each day, leveraging a tool to reason about the ideal type of farming tip to present that day.
- Download PDF Report: Generate and download PDF reports of detection results.

## Style Guidelines:

- Primary color: Fresh green (#6AB04A), symbolizing growth and health in agriculture. Its hex code is generated from an HSL value between green and yellow.
- Background color: Light off-white (#F5F5DC), creating a neutral backdrop to highlight content.
- Accent color: Earthy orange (#D35400), drawing the eye to calls to action. Its hex code is generated from an HSL value roughly 30 degrees away from the primary green on the color wheel. This will make CTAs distinct.
- Body and headline font: 'Inter' (sans-serif) for a clean and modern user interface.
- Use clear and intuitive icons representing plants, diseases, and farming actions.
- Clean and responsive layout with clear sectioning for easy navigation and optimal user experience across devices.
- Subtle animations for feedback during AI processing and transitions between pages.
